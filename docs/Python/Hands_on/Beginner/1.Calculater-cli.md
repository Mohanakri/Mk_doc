## **Step 1 ‚Äî Project Concept**

A **CLI Calculator** is a command-line tool that takes user input for two numbers and an operation, then outputs the result.
We‚Äôll cover:

1. **Basic arithmetic** ‚Äî `+`, `-`, `*`, `/`, `%`, `**`
2. **Error handling** ‚Äî avoid crashes if user inputs text or divides by zero
3. **Clean code** ‚Äî functions for each operation, a dispatcher to choose
4. **Extra features** ‚Äî keep a history of calculations in a file

## **Step 2 ‚Äî Core Concepts**

### 1. **Functions**

We‚Äôll use one function per operation:

```python
def add(x, y): return x + y
def subtract(x, y): return x - y
...
```

This makes code **modular** and easy to expand.

### 3. **Error Handling**

* Division by zero ‚Üí custom message
* Wrong menu choice ‚Üí prompt again

### 4. **History Feature**

* Append each calculation to a `history.txt`
* Option to view history

## **Step 3 ‚Äî Full Code with Explanations**

```python
import os

# --- Operations ---
def add(x, y): return x + y
def subtract(x, y): return x - y
def multiply(x, y): return x * y
def divide(x, y):
    if y == 0:
        return "‚ùå Error: Division by zero!"
    return x / y
def power(x, y): return x ** y
def modulo(x, y): return x % y

# --- Helper: Save to history ---
def save_history(entry):
    with open("history.txt", "a") as f:
        f.write(entry + "\n")

# --- Helper: Show history ---
def show_history():
    if not os.path.exists("history.txt"):
        print("üì≠ No history yet.")
        return
    print("\n--- Calculation History ---")
    with open("history.txt") as f:
        for line in f:
            print(line.strip())
    print("----------------------------\n")

# --- Main Calculator Loop ---
def calculator():
    while True:
        print("\n=== Python CLI Calculator ===")
        print("1. Add (+)")
        print("2. Subtract (-)")
        print("3. Multiply (*)")
        print("4. Divide (/)")
        print("5. Power (**)")
        print("6. Modulo (%)")
        print("7. Show History")
        print("0. Exit")

        choice = input("Enter choice: ")

        if choice == "0":
            print("üëã Goodbye!")
            break
        elif choice == "7":
            show_history()
            continue

        try:
            num1 = float(input("Enter first number: "))
            num2 = float(input("Enter second number: "))
        except ValueError:
            print("‚ö† Please enter valid numbers!")
            continue

        operations = {
            "1": ("+", add),
            "2": ("-", subtract),
            "3": ("*", multiply),
            "4": ("/", divide),
            "5": ("**", power),
            "6": ("%", modulo)
        }

        if choice in operations:
            symbol, func = operations[choice]
            result = func(num1, num2)
            print(f"‚úÖ Result: {result}")
            save_history(f"{num1} {symbol} {num2} = {result}")
        else:
            print("‚ö† Invalid choice!")

if __name__ == "__main__":
    calculator()
```

## **Step 4 ‚Äî How It Works**

3. **Error handling** ‚Äî `try/except` for invalid numbers
4. **History feature** ‚Äî Saves results in `history.txt` and can display them
